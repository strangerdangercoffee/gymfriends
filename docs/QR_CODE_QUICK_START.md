# QR Code Friend Addition - Quick Start Guide

## Installation

Run this command to install the required packages:

```bash
npm install
```

That's it! The packages are already added to `package.json`:
- `expo-camera` (for camera access and scanning)
- `react-native-qrcode-svg` (for generating QR codes)

## How to Use

### For Users: Adding Friends with QR Codes

#### Option 1: Show Your QR Code

1. Open the **Friends** tab
2. Tap **"Show My Code"** button (blue QR code icon)
3. Show your screen to your friend
4. They scan it with their phone
5. You're now friends! 🎉

#### Option 2: Scan a Friend's QR Code

1. Open the **Friends** tab
2. Tap **"Scan QR Code"** button (green scan icon)
3. Grant camera permission if prompted
4. Point your camera at their QR code
5. Code will scan automatically
6. You're now friends! 🎉

## Key Benefits

✅ **No approval needed** - Instant friendship  
✅ **Perfect for gyms** - Quick connections in person  
✅ **No email required** - Just scan and go  
✅ **Mutual friendship** - Both users added automatically  
✅ **Safe** - Can't add yourself or duplicate friends  

## Tips for Best Results

📱 **Keep screen bright** when showing your QR code  
🎯 **Hold phone steady** when scanning  
💡 **Good lighting** helps scanning  
📏 **Optimal distance** - about 6-12 inches away  

## Troubleshooting

### Camera Won't Open?
- Go to Settings → GymFriends → Enable Camera permission
- Close and reopen the app

### QR Code Won't Scan?
- Increase screen brightness
- Try moving closer or farther
- Make sure QR code is flat and visible

### Already Friends Error?
- You're already friends with this person!
- No need to scan again

Super simple way to add workout buddies at the gym. Just scan and go! 💪
